Tampilan Akhir React

Command Prompt


npx create-react-app cat-shop
cd cat-shop
npm install react-router-dom bootstrap
npm start

Langkah-langkah

  1. Install React menggunakan npx create-react-app cat-shop.
  2. Buka folder project dengan cd cat-shop.
  3. Install React Router dan Bootstrap dengan npm install react-router-dom bootstrap.
  4. Jalankan aplikasi dengan npm start.
  5. Modifikasi kode di src/App.js untuk menambahkan navigasi dan halaman tambahan.
  6. Buat halaman Home, Gallery, Contact, dan About.
